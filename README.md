# Used Car Transaction Price Prediction

## Project Overview
This project is based on the Alibaba Tianchi competition data. The goal is to predict the transaction price of used cars based on their features using machine learning techniques.

## Methods
- **Data Cleaning:** Handled missing values and outliers.
- **Feature Engineering:** Created new features based on time series and sales statistics.
- **Model:** Implemented and compared models including Random Forest, XGBoost, and LightGBM.
